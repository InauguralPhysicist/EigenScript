## What Makes EigenScript Unique?

### 1. Self-Interrogation
Your code can ask questions about itself:
```eigenscript
# Traditional code is blind
x is compute of data

if converged:           # "Am I done?"
    return result
if oscillating:         # "Am I stuck?"
    try_different_approach
```

### 2. Geometric Semantics
Every operation has **geometric meaning**:
- Variables are vectors in semantic space
- Operations are transformations in that space
- Convergence is geometric (approaching an eigenstate)
- Types emerge from geometric properties (norm signatures)

### 3. Stable Self-Reference
Functions can reference themselves without exploding:
```eigenscript
define observer as:
    # This converges instead of crashing
    meta is observer of observer
    return meta
```
